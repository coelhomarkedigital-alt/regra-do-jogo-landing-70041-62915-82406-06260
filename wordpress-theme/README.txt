=== Tema Regras do Jogo ===

Instruções de Instalação:

1. Faça o download de todos os arquivos desta pasta
2. No painel do WordPress, vá em Aparência > Temas > Adicionar Novo
3. Clique em "Enviar Tema" e selecione o arquivo ZIP com todos esses arquivos
4. Ative o tema

Arquivos incluídos:
- style.css: Estilos principais do tema
- functions.php: Funções e recursos do tema
- index.php: Template principal
- header.php: Cabeçalho do site
- footer.php: Rodapé do site
- page.php: Template para páginas
- single.php: Template para posts individuais

Shortcodes disponíveis:

[cta_button text="Texto do Botão" url="https://seu-link.com"]
- Cria um botão de call-to-action estilizado

[video src="URL_DO_VIDEO.mp4"]
- Incorpora um vídeo hospedado no seu servidor

Custom Post Type:
- Livros: Para gerenciar os livros do clube

Recursos:
- Suporte a imagens destacadas
- Menus personalizáveis (principal e rodapé)
- Responsivo e otimizado para SEO
- Área de widgets na barra lateral

Para personalizar:
- Vá em Aparência > Personalizar para configurar logo, cores e menus
- Use Aparência > Menus para criar seus menus
- Adicione widgets em Aparência > Widgets
